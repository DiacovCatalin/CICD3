# Laborator3 - CI/CD cu GitHub Actions

Acest proiect demonstrează configurarea unui pipeline CI/CD avansat folosind GitHub Actions.

## Structura proiectului

- `script.py` - Script Python simplu
- `test_eg.py` - Test de bază cu pytest
- `.github/workflows/python-ci.yml` - Workflow GitHub Actions

## Funcționalități CI/CD

- Rulare pe multiple platforme: Ubuntu și Windows
- Testare pe multiple versiuni Python: 3.8, 3.9, 3.10
- Linting automat cu flake8
- Rulare script și teste
- Salvarea rezultatelor ca artefacte
